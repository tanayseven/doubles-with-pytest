Doubles with PyTest
===================

A short introduction to different types of test doubles using pytest
--------------------------------------------------------------------

1. Dummies
2. Mocks
3. Spys
4. Stubs
5. Fakes

Running
-------

```bash
poetry install
poetry run pytest # to test
poetry run python -m sample_app.app
```
